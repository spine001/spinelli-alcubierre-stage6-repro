# Stage 7F Gaussian Width Confirmation at N71

Stage 7F tested whether the Gaussian-pulse width trend observed at N61 survives higher resolution.

Cases tested:

- geometry: gaussian_pulse
- N: 71
- amplitude: 0.025
- widths: 1.25, 1.50, 1.75, 2.00

Main findings:

1. The N71 run confirms the N61 Gaussian width trend.
2. As width increases, Action/Fit moves toward unity.
3. As width increases, beta_fit moves toward zero, not toward -1.
4. Bianchi residual remains small and stable.
5. The result supports the interpretation that beta=-1 is specific to steep wall-localized geometries such as Alcubierre, while smoother Gaussian curvature pulses produce geometry- and scale-dependent beta values.

Interpretation:

The action-derived correction tensor remains predictive outside Alcubierre geometry, but the fitted beta coefficient is not universal. The Gaussian-pulse result strengthens the cross-geometry evidence for the framework while requiring a more nuanced physical interpretation.
