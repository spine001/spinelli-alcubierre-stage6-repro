# Stage 8E PG-Horizon Very-High-Resolution Summary

Stage 8E extended the PG-horizon exterior near-horizon branch to N131 and N141.

Cases tested:

- geometry: pg_horizon
- model: Painleve-Gullstrand-like horizon-regular radial-flow metric
- mask: schwarz_exterior
- mass: 1.25
- eps: 0.35
- width: 1.00
- N: 131, 141

Main findings:

1. Action/Fit remained essentially pinned to unity through N141.
2. Relative tensor difference remained below 0.5%.
3. N141 produced the best high-resolution tensor difference so far, approximately 0.301%.
4. Absolute Bianchi continued to decrease with increasing resolution.
5. Relative Bianchi also continued to decrease, reaching approximately 4.60% at N141.
6. beta_fit remained negative and bounded, though not monotonic.
7. The result survived severe memory and swap stress, strengthening confidence that the PG-horizon exterior branch is not a low-resolution artifact.

Interpretation:

Stage 8E establishes the PG-horizon exterior branch as a very-high-resolution near-horizon validation case for the Spinelli Framework. The action-derived correction tensor remains strongly predictive in a horizon-regular radial-flow chart. The fitted beta coefficient behaves as an effective geometry-dependent parameter rather than a universal scalar constant.
